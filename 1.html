HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language forcreating web pages and web applications.web browsers receive HTML documents from a web server or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. with html constructs, images and other pages, such as interactive forms, may be embedded into the ad rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other item. html elements are delineated by tags, written using angle brackets.
